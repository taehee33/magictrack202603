# MagicTrack v1.1.0

MagicTrack는 맥북 내장 트랙패드와 블루투스 매직 트랙패드의 설정을 분리 저장하고, 원하는 시점에 빠르게 적용할 수 있게 만든 macOS 앱입니다.

이번 버전은 장치별 설정 저장, 수동 적용, 빠른 전환, 프리셋 관리, 설정 구조 정리에 초점을 맞췄습니다.

## 포함 기능

- 내장 트랙패드 / 매직 트랙패드 설정 분리 저장
- 이동 속도 / 스크롤 속도 / 클릭 압력 조절
- 장치별 수동 적용
- 프리셋 저장 및 적용
- 앱 자동전환 규칙 관리
- 실행 중 Dock / 메뉴바 표시 제어
- 전역 단축키 `option + M` 빠른 전환

## 현재 제한 사항

- 매직 트랙패드 회전은 현재 실험 기능입니다
- Apple 트랙패드의 실입력 감지, 좌표/제스처 감지, 입력 기반 자동화는 현재 미지원입니다
- Apple 플랫폼 제약으로 완전한 실시간 독립 감도 제어에는 제한이 있습니다

## 권한

일부 기능에는 다음 권한이 필요할 수 있습니다.

- 입력 모니터링
- 손쉬운 사용
