# MagicTrack v1.1.0 사용자 공지문

아래 문안은 X, 블로그, 커뮤니티, 오픈채팅, Discord, 카카오톡 공지 등에 바로 올릴 수 있는 공개 공지문 초안입니다.

## 짧은 공지문

```text
MagicTrack v1.1.0을 공개합니다.

맥북 내장 트랙패드와 블루투스 매직 트랙패드의 설정을 분리 저장하고, 원하는 시점에 빠르게 적용할 수 있는 macOS 앱입니다.

- 장치별 설정 저장
- 이동 속도 / 스크롤 속도 / 클릭 압력 조절
- 프리셋 저장 및 적용
- 앱 자동전환 규칙 관리
- option + M 빠른 전환
- 실행 중 Dock / 메뉴바 표시 제어

다운로드와 설치 방법은 저장소 문서를 참고해주세요.

주의:
현재 배포본은 Apple Developer ID 서명 및 notarization이 적용되지 않았을 수 있어, 처음 실행 시 macOS 보안 경고가 나타날 수 있습니다.
```

## 긴 공지문

```md
MagicTrack v1.1.0을 공개합니다.

MagicTrack는 맥북 내장 트랙패드와 블루투스 매직 트랙패드의 설정을 분리 저장하고, 원하는 시점에 빠르게 적용할 수 있게 만든 macOS 앱입니다.

### 주요 기능

- 내장 / 매직 트랙패드 설정 분리 저장
- 이동 속도 / 스크롤 속도 / 클릭 압력 조절
- 장치별 수동 적용
- 프리셋 저장 및 적용
- 앱 자동전환 규칙 관리
- option + M 빠른 전환
- 실행 중 Dock / 메뉴바 표시 제어

### 참고할 점

- 일부 기능은 Apple 플랫폼 제약으로 제한될 수 있습니다
- 매직 트랙패드 회전은 현재 실험 기능입니다
- Apple 트랙패드의 실입력 감지, 좌표/제스처 감지, 입력 기반 자동화는 현재 미지원입니다

### 설치 관련 안내

현재 배포본은 Apple Developer ID 서명 및 notarization이 적용되지 않았을 수 있습니다.
이 경우 처음 실행 시 macOS 보안 경고가 나타날 수 있습니다.

문제가 발생하면 설치 가이드를 참고해주세요.
```

```md
## MagicTrack v1.1.0 배포

macOS용 트랙패드 설정 앱 MagicTrack v1.1.0을 배포합니다.

지원 기능:
- 내장/매직 트랙패드 설정 분리 저장
- 이동 속도 / 스크롤 속도 / 클릭 압력 조절
- 프리셋 저장 / 적용
- 앱 자동전환 규칙
- option + M 빠른 전환
- Dock / 메뉴바 표시 설정

제한 사항:
- Apple 트랙패드의 실입력 감지 / 좌표·제스처 감지 / 입력 기반 자동화는 현재 미지원
- 매직 트랙패드 회전은 실험 기능

설치 시 macOS 보안 경고가 나타날 수 있으니, 설치 가이드를 함께 확인해주세요.
```

## 공지문 작성 시 주의

- 실제 다운로드 URL은 업로드 후 넣기
- `Developer ID`, `Team ID`, 개인 이메일 같은 정보는 넣지 않기
- `미지원` 기능은 과장하지 않기
- 설치 가이드 링크를 함께 제공하기
