# MagicTrack v1.1.0 릴리즈 노트

## 개요

MagicTrack v1.1.0은 내장 트랙패드와 매직 트랙패드의 설정을 분리 저장하고, 사용자가 원하는 시점에 해당 설정을 빠르게 적용하는 데 초점을 맞춘 버전입니다.

이번 버전에서는 기능 구조와 UI를 정리했고, 수동 적용 중심 사용 흐름을 강화했습니다.

## 추가/개선된 기능

### 장치별 설정 관리

- 내장 트랙패드 / 매직 트랙패드 설정 분리 저장
- 이동 속도 / 스크롤 속도 / 클릭 압력 조절
- 각 장치 카드에서 현재 설정 즉시 적용

### 빠른 전환

- `option + M` 전역 단축키 추가
- 내장 / 매직 트랙패드 빠른 전환 지원

### 프리셋

- 현재 설정을 프리셋으로 저장
- 저장된 프리셋 불러오기 및 적용

### 설정 구조 개선

- 메인 탭의 `앱 자동전환`을 `설정` 탭으로 재구성
- 실행 중 `Dock에 보이기` 토글 추가
- 실행 중 `메뉴바에 보이기` 토글 추가
- Dock / 메뉴바 둘 다 사라져 앱 접근이 끊기지 않도록 보호 로직 추가

### 앱 자동전환

- 앱별 규칙 추가/삭제 UI 정리
- 설정 탭 안에서 자동전환 활성화와 규칙 관리 가능

### 배포 준비

- 버전 정보 표기 개선
- Release zip / DMG 패키지 스크립트 추가
- 배포 문서 초안 추가

## 실험 기능

다음 기능은 현재 실험 또는 제한 상태입니다.

- 매직 트랙패드 회전
- Apple 트랙패드 실입력 감지
- 좌표/제스처 감지
- 입력 기반 자동화

이 항목은 UI에 표시될 수 있으나, 현재 버전에서 완전한 동작을 보장하지 않습니다.

## 알려진 제한 사항

- Apple 플랫폼 제약으로 내장/매직 트랙패드의 완전한 실시간 독립 이동 속도 제어는 제한됩니다
- 입력 기반 자동 감지 대신 수동 적용 중심 사용을 권장합니다
- 인증서 및 notarization이 없는 배포본은 macOS 보안 경고가 발생할 수 있습니다

## 권장 사용 방식

1. 각 장치별 설정을 먼저 저장
2. 필요 시 카드의 `이 설정 적용` 사용
3. 자주 쓰는 조합은 프리셋으로 저장
4. 빠른 전환은 `option + M` 사용

## 업데이트 시 확인할 것

- 입력 모니터링 권한
- 손쉬운 사용 권한
- Dock / 메뉴바 표시 설정
- 프리셋과 앱 자동전환 규칙 상태

## 배포 메모

현재 버전은 대중 공개 전 단계에서 다음이 남아 있습니다.

- Apple Developer Program 가입
- Developer ID Application 인증서 발급
- notarization 적용
- 서명된 DMG 재배포
