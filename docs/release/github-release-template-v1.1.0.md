# GitHub Release Template: v1.1.0

아래 본문은 GitHub Releases에 바로 붙여넣을 수 있는 공개 배포용 템플릿입니다.

제목 예시:

```text
MagicTrack v1.1.0
```

본문 예시:

```md
## MagicTrack v1.1.0

MagicTrack는 맥북 내장 트랙패드와 블루투스 매직 트랙패드의 설정을 분리 저장하고, 원하는 시점에 빠르게 적용할 수 있게 만든 macOS 앱입니다.

### 포함 기능

- 내장/매직 트랙패드 설정 분리 저장
- 이동 속도 / 스크롤 속도 / 클릭 압력 조절
- 장치별 수동 적용
- 프리셋 저장 및 적용
- 앱 자동전환 규칙 관리
- 실행 중 Dock / 메뉴바 표시 제어
- option + M 빠른 전환

### 현재 제한 사항

- 매직 트랙패드 회전은 현재 실험 기능입니다
- Apple 트랙패드의 실입력 감지, 좌표/제스처 감지, 입력 기반 자동화는 현재 미지원입니다
- Apple 플랫폼 제약으로 완전한 실시간 독립 감도 제어에는 제한이 있습니다

### 권한

일부 기능에는 다음 권한이 필요할 수 있습니다.

- 입력 모니터링
- 손쉬운 사용

### 설치 파일

- `MagicTrack-x.y.z.dmg`
- `MagicTrack-x.y.z-macOS.zip`

### 설치 전 주의

이 배포본은 아직 Apple Developer ID 서명 및 notarization이 적용되지 않았을 수 있습니다.
처음 실행 시 macOS 보안 경고가 나타날 수 있습니다.
문제가 발생하면 저장소의 설치 가이드를 참고해주세요.

### 설치 가이드

- 한국어: `docs/install-guide-ko.md`
- English: `docs/install-guide-en.md`
```

## 업로드 전 체크

- 버전 번호가 실제 앱 버전과 같은지
- 파일명이 실제 산출물과 같은지
- 제한 사항 문구가 현재 앱 상태와 맞는지
- 개인 이메일, 팀 ID, 로컬 경로가 본문에 없는지
